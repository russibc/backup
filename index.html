Epibuilder 2.0

A modular web platform for assembling, predicting, and analyzing B-cell linear epitopes

Learn More

About

Epibuilder 2.0 is a Dockerized, modular web platform developed by the Bioinformatics Lab at UFSC for in silico analysis of linear B-cell epitopes. It integrates BepiPred 3.0 and BLAST, and supports GPU or CPU environments. Users can submit .fasta or .csv files and interact with results directly via a web interface.

The platform was developed as part of a master's dissertation in Biotechnology and Biosciences at UFSC, aiming to improve usability and reproducibility for immunoinformatics workflows.

Usage

Epibuilder 2.0 provides three interfaces:

  • Web interface (Angular 12) for data submission, analysis tracking, and visualization
  • Command-line interface via Docker container
  • Workflow automation using Nextflow

Functionality includes:

  • Prediction of B-cell linear epitopes using BepiPred 3.0
  • Topological and biochemical characterization (Emini, Karplus-Schulz, Chou-Fasman, etc.)
  • Similarity searches using BLAST against curated IEDB datasets
  • Role-based authentication (admin/user) and full result history tracking

Repositories

The system is composed of the following repositories, all available at GitHub:

  • epibuilder-frontend: Angular 12-based GUI
  • epibuilder-backend: Spring Boot API service
  • epibuilder-core: Command-line processing engine
  • epibuilder-deploy: Docker + CI/CD setup
  • Nextflow scripts: included in backend for workflow orchestration

Requirements

  • Docker installed on your system.

Running the Docker Containers

Debian-based Docker Container

To run the EPIBuilder Docker container with a Debian base, use the following command:

docker run -it \
                      -p 80:80 \
                      -p 8080:8080 \
                      -v /home/bioinfo/github/epibuilder-deploy/pipeline:/pipeline \
                      bioinfoufsc/epibuilder:debian-cpu \
                      /bin/bash

Ubuntu-based Docker Container

To run the EPIBuilder Docker container with an Ubuntu base, use the following command:

docker run -it \
                      -p 80:80 \
                      -p 8080:8080 \
                      -v /home/bioinfo/github/epibuilder-deploy/pipeline:/pipeline \
                      bioinfoufsc/epibuilder:ubuntu-gpu \
                      /bin/bash

Accessing the Web Interface

Once the container is running, you can access the web interface by opening your browser and navigating to: 

http://localhost

Docker Images

To pull the Docker image for the appropriate environment, use the following commands:

For the Ubuntu GPU version:

docker pull bioinfoufsc/epibuilder:ubuntu-gpu

For the Debian CPU version:

docker pull bioinfoufsc/epibuilder:debian-cpu

To install locally, clone the repositories and follow the README instructions for Docker or development environments.
